# How to implement content efficiently

Have devs structure content like:

```
content/
  newsfeed.json
  characters.json
  events_whimsy.json
  nicknames.json
  achievements.json
  headlines.json
  citizen_quotes.json
```

This lets you expand pedagogy without touching engine code.
